# RandomNumbers.MersenneTwisters.MT19937 — Type.

MT19937 <: MersenneTwister{UInt32}
MT19937([seed])

MT19937 RNG. The seed is a Tuple of 624 UInt32 numbers, or an Integer which will be automatically convert to an UInt32 number.
